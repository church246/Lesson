第一課：這到底是誰？

風暴的平靜者

經文：馬可福音 4:35–41

一、聖經密碼:

35. 當那天晚上，耶穌對門徒說：『我們渡到那邊去罷。』
36. 門徒離開眾人，耶穌仍在船上，他們就把祂一同帶去；也有別的船和祂同行。
37. 忽然起了暴風，波浪打入船內，甚至船要滿了水。
38. 耶穌在船尾上，枕著枕頭睡覺；門徒叫醒了祂，說：『夫子，我們喪命，你不顧麼？』
39. 耶穌醒了；斥責風，向海說：『住了罷，靜了罷。』風就止住，大大的平靜了。
40. 耶穌對他們說：『為甚麼膽怯，你們還沒有信心麼？』
41. 他們就大大的懼怕，彼此說：『這到底是誰？連風和海也聽從祂了。』

請先閱讀聖經經文，然後依序回應下列問題與反思。

密碼試解

我們不曉得到底那是哪一個月黑風高的夜晚，但是讓我們仔細地研究一下，這段聖經我們可以找到甚麼可以學到的武功秘笈。

  1. 當天晚上發生甚麼事情讓門徒十分有壓力？他們採取甚麼方法處理這個危機？依你之見，這個方法恰當嗎？

  2. 耶穌處理暴風的方法是甚麼？若你在船上看到這種情況、聽到耶穌的說法，你會有甚麼反應？

  3. 等到耶穌處理完整個事件，祂對門徒說了甚麼？你認為祂為什麼會這樣說？

  4. 從最後門徒的反應來看，原先他們以為耶穌是誰？這樣的事件讓門徒對耶穌的認識有甚麼突破？

三、經文解碼

相信我們多半有坐船的經驗，喜歡嗎？

這裡讓我們看到一群人在海上精彩的經歷。您一定注意到了，這群人是耶穌的學生；但是我們若追查他們在成為門徒之前的工作，就會發現他們當中有好些人原先的職業是漁夫！漁夫在半夜開船是駕輕就熟的事，所以當耶穌說「我們渡到那邊去」的時候，沒有人有甚麼意見。然而那天晚上他們卻碰上了讓他們終生難忘的事。

這段聖經告訴我們，暴風和大浪是忽然起的，完全沒有預警，而且這樣的情況非常的凶險。您注意到了嗎？這一群有經驗的漁夫跑去找誰？

沒錯，他們竟然跑去找一位木匠出身的人幫忙！有人說這是看出門徒們的信心，可是若我們深入一點研究，就會發現：門徒真的是找耶穌幫忙嗎？

門徒們雖然叫醒耶穌，並且一副「你不顧，我們會死」的態勢，可是他們叫耶穌甚麼？

還有，等耶穌解決問題之後，他們的反應又是甚麼？

可見，他們其實並沒有指望耶穌真能幫上什麼忙；或許只是想讓耶穌起來幫忙舀水，或是他們甚至連這都不敢想，叫耶穌純粹是不爽他居然還在那裏高枕無憂，就算他幫不上忙，也不能個人在那裏納涼啊!

然而我們看見就算人不曉得, 不相信耶穌能幫甚麼忙, 只要我們來到他面前, 所得的總是超過所求所想. 讓我們整理一些要點：
　

  1. 如果我們去瞭解這個事件發生的時間, 大約是耶穌已經呼召十二個門徒之後, 在此之前,　耶穌曾醫治彼得的岳母, 潔淨長大痲瘋的人, 治好了所有來求診的人, 而且趕出污鬼， 雖然門徒們並不是第一次看耶穌行神蹟. 然而以前那些神蹟, 對他們而言或許只是讓他們認識耶穌是一位偉大的醫生, 很多人都可以辦到的. 他們並沒有全面的認識這一位是真神.
  2. 我們會發現, 即使門徒當中有人是漁夫, 熟悉大海的各種狀況, 還是會有無法控制的情形. 我們常會說：人定勝天, 但是天災卻是人無法控制！我們不得不承認：大自然是渺小的人類所無法掌控的. 有些時候我們以為自己能夠處理, 甚至駕輕就熟的事情, 卻成為我們的致命傷. 這種情形也會發生在我們的日常生活中, 有時我們會感嘆, 甚至連三餐都搞不定！天下還是有許多難事, 連有心人都無法處理, 因此我們不得不承認自己無法處理人生的每個狀況, 必須求助或尋求外援.
  3. 到底耶穌若與我們同在, 會帶來我們人生多麼大的不同？大概是我們無法想像的. 我們以為只是有一個信仰, 有一個安慰, 卻從沒想到祂會在每一個意外的狀況中, 真實的保護我們. 門徒的反應懼怕. 因為其一, 他們發現耶穌跟他們認知的上帝是不一樣的. 我們所信的這位上帝是沒法子被任何人操控的, 因為祂不像人手所造民間信仰的那些像, 若按我們所求, 我們才拜它. 當然另一個壓力點是剛才曾經那麼兇的對一位可以斥責風海的對象說話, 下場會如何呢？令人驚奇是耶穌並沒有在這件事上發什麼脾氣, 只是問了一句：　你們還沒有信心嗎？這位全能的主宰真是溫柔恩典的神啊!
  4. 大大的懼怕! 人與真神的相會, 難免帶來生命的震撼! 這到底是誰? 則是每位求道者都想要明白的. 耶穌是大自然風暴的平靜者, 也是這一群門徒心中風暴的平靜者, 使他們從驚恐面對生命的危險, 到回頭思想生命的真相, 但願我們都能經歷生命風暴被主撫平, 也能尋到生命真相問題的答案.

要點整理

  1. 若看事件發生的時間，這大約是在耶穌已呼召十二門徒之後。此前祂曾醫治彼得的岳母、潔淨長大痲瘋的人、醫治許多病人並趕出污鬼。門徒不是第一次看見神蹟，但先前的經歷多半讓他們把耶穌視為偉大的醫治者，尚未全面認識祂是真神。
  2. 即使門徒中有人是漁夫、熟悉海象，仍有無法控制的情形。人常說「人定勝天」，然而天災與許多變故並非人力所能掌控；我們終究需要求助與外援。
  3. 耶穌與我們同在，帶來的改變往往超過想像。我們以為信仰只是安慰，卻忽略祂在意外中也能真實保護。門徒懼怕，一方面因為他們發現耶穌超越他們對上帝的想像；另一方面，也因為剛才他們對這位能斥責風海者的口氣顯得無禮。然而耶穌並未發怒，只是說：「你們還沒有信心嗎？」顯明祂是溫柔、恩典的主。
  4. 人與真神相遇，常帶來生命的震撼（「他們就大大的懼怕」）。「這到底是誰？」是每位求道者都要回應的問題。耶穌是大自然風暴的平靜者，也是人心風暴的平靜者，使人從驚恐轉向思索生命真相。

四、運用密碼:

  1. 讀完這段聖經，想想看自己對耶穌的認識是甚麼？如果那天你在船上，你會向耶穌求助嗎？若會，你求助的動機是甚麼？

  2. 最近你的生命中有甚麼風暴讓你難過痛苦嗎？不妨寫下來，讓我們把它告訴耶穌，經歷祂平靜你生命的風暴！

同心禱告（可參考）

主耶穌，當我的生命遭遇風浪時，求你與我同在。教我在害怕時依靠你，讓你的話語平靜我的心，使我學會信靠、學會交託。奉主名求，阿們。

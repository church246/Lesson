第四課：靈界的管轄者

經文：路加福音 8:26–39

26. 他們到了格拉森人的地方，就是加利利的對面。
27. 耶穌上了岸，就有城裏一個被鬼附著的人，迎面而來；這個人許久不穿衣服， 不住房子，只住在墳塋裏。
28. 他見了耶穌，就俯伏在祂面前，大聲喊叫，說：『至高神的兒子耶穌，我與你 有甚麼相干？求你不要叫我受苦。』
29. 是因耶穌曾吩咐污鬼從那人身上出來；原來這鬼屢次抓住他，他常被人看守， 又被鐵鍊和腳鐐捆鎖，他竟把鎖鍊掙斷，被鬼趕到曠野去。
30. 耶穌問他說：『你名叫甚麼？』他說：『我名叫群。』這是因為附著他的鬼多。
31. 鬼就央求耶穌，不要吩咐牠們到無底坑裏去。
32. 那裏有一大群豬，在山上喫食；鬼央求耶穌，准牠們進入豬裏去；耶穌准了牠 們。
33. 鬼就從那人出來，進入豬裏去；於是那群豬闖下山崖，投在湖裏淹死了。
34. 放豬的看見這事就逃跑了，去告訴城裏和鄉下的人。
35. 眾人出來要看是甚麼事；到了耶穌那裏，看見鬼所離開的那人，坐在耶穌腳前， 穿著衣服，心裏明白過來，他們就害怕。
36. 看見這事的，便將被鬼附著的人怎麼得救，告訴他們。
37. 格拉森四圍的人，因為害怕得很，都求耶穌離開他們；耶穌就上船回去了。
38. 鬼所離開的那人，懇求和耶穌同在；耶穌卻打發他回去，
39. 說：『你回家去，傳說神為你作了何等大的事。』他就去滿城裏傳揚耶穌為他 作了何等 大的事。

建議先朗讀經文，再依序回應導讀問題與反思題，最後以禱告回應。

導讀問題

  1. 在這段經文中，描述有一個人他的行為十分奇特。請整理：他有哪些與常人不同之處？這個人叫什麼名字？

  2. 你是否注意到：當耶穌與這個人說話時，是誰在回答？這顯出什麼問題？

  3. 耶穌如何處理這個人的問題？從經濟或社會角度看，這樣的處理划算嗎？那地方的人有何看法？

  4. 這件事最後的結局是什麼？若你是當事人或其親友，你會有何感受？

經文解碼（背景與觀察）

這個事件恰好發生在耶稣平靜海上風暴之後。門徒驚魂甫定，剛下船便見識到耶穌在靈界的權柄。試著站在門徒位置，一起觀察：

1. 迎面來的人

一個人迎面而來，但他像人卻又失去人的樣式：沒穿衣服、居住在墳堆、身上有弄斷的鎖鏈、力大難以制伏，衛生與氣味不堪。這樣的人使人本能迴避、恐懼，也讓人懷疑：誰能處理他？

2. 主動的救助者

敘事採用倒述：先記錄污鬼的反應，後補充耶穌已經吩咐污鬼離開。這顯示耶穌不僅是物質世界的主（能止息風海），也是靈界的主（污鬼畏懼祂的權柄）。污鬼懼怕提早受審，求耶穌准許；耶穌按著上帝時間表行事，也顧念一個被轄制的人的尊嚴與自由。

3. 值得嗎？

被問「你名叫什麼？」時，回答者其實是污鬼，顯出此人自我主體被奪的悲慘。耶穌看重一個人的價值，超過一大群豬或一地的經濟損失；然而當地人因恐懼與不願改變而請耶穌離開。你如何看待「一個靈魂」與「群體利益」的張力？

4. 完美的結局

那人恢復清醒、坐好穿衣，成為有見證能力的人，被差回家鄉述說主的恩典。耶穌以此事顯明：任何靈界勢力都不能在祂面前站立得住；祂能轉化我們對黑暗與靈異的恐懼，成為對主的敬畏與信靠。

要點整理

  1. 耶穌不僅平靜風浪，也是靈界的管轄者；祂的權柄使污鬼戰兢。
  2. 在人看來「無可救藥」的人，在主裡可以完全恢復—理智、尊嚴、關係與使命。
  3. 主看重一個人的價值，超過可衡量的經濟利益；愛常伴隨代價。
  4. 群眾可能因害怕改變而拒絕耶穌；門徒則被呼召洞察並傳揚主的作為。

個人反思

  1. 你的人生是否經歷過與靈界的接觸？你的感受是什麼？當時如何處理？

    範例回應：「我有接觸一些宗教行為，但承蒙神保守，除了經歷上帝的大能外，沒有經歷其他靈界的事。」

  2. 耶穌能拯救並保護我們不被不好的靈界勢力壓迫。請提出你的需要，向祂傾心吐意；讓我們一起禱告。

同心禱告（可參考）

主耶穌，祢是掌管萬有與靈界的主。求祢釋放被轄制的人，使我們在祢裡面得自由、恢復與盼望。當我害怕時，讓我投靠祢的大能與慈愛；也使我願意見證祢在我身上的作為。奉主名求，阿們。

帶領建議：流程可採「導讀問題 → 經文解碼 → 要點整理 → 個人反思 → 禱告回應」。若用於小組，可分組討論第 3 題的「值得嗎？」以連結信仰與公共議題。

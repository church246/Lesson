第四課：靈界的管轄者

一、聖經密碼：路加福音 8:26–39

26. 他們到了格拉森人的地方，就是加利利的對面。
27. 耶穌上了岸，就有城裏一個被鬼附著的人，迎面而來；這個人許久不穿衣服，不住房子，只住在墳塋裏。
28. 他見了耶穌，就俯伏在祂面前，大聲喊叫，說：『至高神的兒子耶穌，我與你有甚麼相干？求你不要叫我受苦。』
29. 是因耶穌曾吩咐污鬼從那人身上出來；原來這鬼屢次抓住他，他常被人看守， 又被鐵鍊和腳鐐捆鎖，他竟把鎖鍊掙斷，被鬼趕到曠野去。
30. 耶穌問他說：『你名叫甚麼？』他說：『我名叫群。』這是因為附著他的鬼多。
31. 鬼就央求耶穌，不要吩咐牠們到無底坑裏去。
32. 那裏有一大群豬，在山上喫食；鬼央求耶穌，准牠們進入豬裏去；耶穌准了牠們。
33. 鬼就從那人出來，進入豬裏去；於是那群豬闖下山崖，投在湖裏淹死了。
34. 放豬的看見這事就逃跑了，去告訴城裏和鄉下的人。
35. 眾人出來要看是甚麼事；到了耶穌那裏，看見鬼所離開的那人，坐在耶穌腳前，穿著衣服，心裏明白過來，他們就害怕。
36. 看見這事的，便將被鬼附著的人怎麼得救，告訴他們。
37. 格拉森四圍的人，因為害怕得很，都求耶穌離開他們；耶穌就上船回去了。
38. 鬼所離開的那人，懇求和耶穌同在；耶穌卻打發他回去，
39. 說：『你回家去，傳說神為你作了何等大的事。』他就去滿城裏傳揚耶穌為他作了何等大的事。

建議先朗讀經文，再依序回應導讀問題與反思題，最後以禱告回應。

二、密碼試解:

  1. 在這段經文中，描述有一個人他的行為十分奇特。請整理：他有哪些與常人不同之處？這個人叫什麼名字？

  2. 你是否注意到：當耶穌與這個人說話時，是誰在回答？這顯出什麼問題？

  3. 耶穌如何處理這個人的問題？從經濟或社會角度看，這樣的處理划算嗎？那地方的人有何看法？

  4. 這件事最後的結局是什麼？若你是當事人或其親友，你會有何感受？

三、聖經解碼

這個事件恰好發生在耶穌平靜海上的風暴之後, 門徒們驚魂甫定, 剛下船, 就見識到耶穌另一方面的能力. 讓我們置身門徒當中來看看這件事, 相信會有出人意表的發現.

    迎面來的人:讓我們想一想, 一個人迎面而來, 可是他真的是人嗎? 他沒穿衣服, 所有的女士們, 這時候都會自動把頭轉開, 但是別弄錯, 他並不是暴露狂, 他還有其他更讓人有壓力的表現. 他住在墳堆裡, 身上還有弄斷的鎖鏈, 至於他身上的味道呢! 那就甭說了!你想他乾不乾淨? ___________. 哦! 別鬧了, 誰有辦法抓他去洗澡啊! 相信任何人看到他都會自動迴避. 誰有那麼大的力氣可以對付他或者制伏他呢?　我們還是閃一邊吧！ 如果我在耶穌的身邊, 保證會去拉拉耶穌的衣角, 跟他說：老師啊！我們走那邊. 真的, 能處理受超靈界控制的人嗎? 不值得去為一個不像人的人費心, 何況這個人也沒來求, 也沒有他的親人拜託, 耶穌是大忙人, 該做的事很多, 不必理會這種人了！

  1. 一個人迎面而來，但他像人卻又失去人的樣式：沒穿衣服、居住在墳堆、身上有弄斷的鎖鏈、力大難以制伏，衛生與氣味不堪，使人本能迴避、恐懼。

    2. 主動的救助者：聖經記載這個人對耶穌說的話, 值得留心的是, 好像出聲說話的並不是人, 而且聖經採用了倒述法, 先說污鬼的反應, 後來才說, 那是因為耶穌說了一句他們無法承受的話. 耶穌說了甚麼呢?　是的, 耶穌要控制那人的靈界東西離開那個人. 在前二十年, 我們常要證明這世界並不是只有眼睛看得到的東西. 但是現在, 幾乎所有的人都認同, 在這個世界以外, 還有一個靈界. 我們無法不承認我們的生活中充滿了超自然的事. 我們每一個人的一生, 多多少少都會經歷一兩件奇怪的靈異事件. 只是我們不一定會去追究. 聖經的立場十分平衡, 不但清楚的告訴我們, 人是有靈魂的, 也告訴我們, 有一種靈界的勢力是壞的. 這和中國人所說祖先的靈並不相同. 因為如果只是死去祖先的靈就沒有甚麼可怕, 活著還多點威脅, 死了連打人都不會痛了呀！　因此真正能威脅我們的, 一定是原先就超越我們能力的靈. 這裡所說的污鬼, 就是這種, 他能控制人到失去自我的地步. 形容他們是污鬼真是恰如其份. 您看當人被污鬼控制了以後, 就變成一個很髒的人. 傷害自己也失去一切作為人的尊嚴. 耶穌不忍坐視這樣的事情, 他主動地去處理, 污鬼的反應是恐懼, 他們擔心的是聖經上所預言世界結局的事提早發生在他們身上. 他們要求不要提早接受審判, 耶穌接受了他們的請求, 因為的確, 到無底坑去不是現在, 而是在世界結束的時候. 從這裡我們可以發現, 耶穌不但是物質世界的控制者, 也是靈界的掌管者. 這位無所不能的上帝, 卻在意一個小小的人, 一個連親朋都沒有的人.

  2. 敘事採用倒述：先記錄污鬼的反應，後補充耶穌已經吩咐污鬼離開。耶穌不僅是物質世界的主，也是靈界的主。祂顧念被轄制者的尊嚴與自由。

    3. 值得嗎？當耶穌問這個被鬼附著的人『你名叫甚麼』的時候, 他不但沒法子回答, 甚至他的身體成為污鬼的工具, 替污鬼說話. 這是人生最可悲的事情, 完全失去自主的能力. 然而, 當我們看到這樣的人時, 多半不會產生憐憫的情緒, 我們避之唯恐不及. 耶穌卻在意這個人超過一大群豬的價值. 甚至願意為了救他, 賠上在那地方傳道的機會. 值得嗎?　 如果您看見格拉森地方的人, 他們的反應是懼怕的話, 您就可以瞭解, 有些人其實最怕的就是改變, 即使這樣的改變會建立人生的價值. 從別人的角度來看, 或許這人的死活算不了甚麼, 但從愛他的人來看, 他們一定費盡了心思, 用盡了氣力, 最後終於放棄. 如今, 這真的意外的驚喜. 對當事人來說, 又有甚麼比真正自由的活著, 不被轄制重要呢？　嗯！值不值得, 就要看您是誰和您的價值觀了!

  3. 耶穌看重一個人的價值，超過一大群豬或一地的經濟損失；當地人因恐懼與不願改變而請耶穌離開。

    4. 完美的結局：一個似乎是無可救藥的人, 恢復神智, 完全清醒過來, 一個無法控制, 總是傷害自己, 有暴力問題的人, 安安靜靜的坐著, 一個人生已經失去意義, 沒有價值也沒有未來的人, 成為有能力的見證人！注意到了嗎？　他竟然可以滿城裡去傳揚在他身上發生的事. 他成為一個有能力的人. 喂！這結局還不完沒嗎？然而更完美的是, 耶穌用這件事件讓我們瞭解, 任何靈界的勢力, 都無法站立在祂面前. 我們是從小被大人嚇慣了的孩子, 我們怕黑, 我們怕暗夜的聲響, 甚至長大後, 我們也會畏懼一些靈異的事, 耶穌若進到我們的生活中, 祂將改變這一切. 因為祂是真正靈界得掌控者.

  4. 那人恢復清醒、坐好穿衣，成為有見證能力的人，被差回家鄉述說主的恩典。耶穌顯明：任何靈界勢力都不能在祂面前站立得住。

四、密碼運用:

  1. 你的人生是否經歷過與靈界的接觸？你的感受是什麼？當時如何處理？

    範例回應：「我有接觸一些宗教行為，但承蒙神保守，除了經歷上帝的大能外，沒有經歷其他靈界的事。」

  2. 耶穌能拯救並保護我們不被不好的靈界勢力壓迫。請提出你的需要，向祂傾心吐意；讓我們一起禱告。

同心禱告（可參考）

主耶穌，祢是掌管萬有與靈界的主。求祢釋放被轄制的人，使我們在祢裡面得自由、恢復與盼望。當我害怕時，讓我投靠祢的大能與慈愛；也使我願意見證祢在我身上的作為。奉主名求，阿們。

帶領建議：流程可採「導讀問題 → 經文解碼 → 要點整理 → 個人反思 → 禱告回應」。若用於小組，可分組討論第 3 題的「值得嗎？」以連結信仰與公共議題。
